# 🌌 Prisma Bloom Effect

An interactive HTML5 Canvas visual experience featuring a mesmerizing portal vortex, colorful bloom effects, dynamic ripples, and smooth touch-responsive animations. Designed for both desktop and mobile devices, the animation creates an immersive sci-fi energy portal effect.

## ✨ Features

* 🌌 Animated vortex portal effect
* 🌈 Dynamic color-shifting bloom visuals
* 💫 Smooth rotating spiral patterns
* 📱 Touch and mouse interaction support
* 🌊 Ripple effects on click and touch
* ⚡ Real-time canvas rendering
* 🖥️ Responsive fullscreen display
* 🎨 Pure HTML, CSS, and JavaScript

```text

```

## 🎮 How to Use

### Desktop

* Click anywhere to create energy ripples.
* Hold and move the mouse to influence the vortex position.

### Mobile

* Tap the screen to generate ripple waves.
* Touch and drag to interact with the portal animation.
